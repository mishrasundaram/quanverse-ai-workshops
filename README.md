# QuanVerse AI Mastery Website

A static GitHub Pages website for the course:

**QuanVerse AI Mastery: ChatGPT, Codex & Cybersecurity**

## Files

- `index.html`
- `style.css`
- `script.js`

## How to host on GitHub Pages

1. Create a new GitHub repository.
2. Upload these three files to the repository root.
3. Go to repository `Settings`.
4. Open `Pages`.
5. Under `Build and deployment`, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Save.
7. Your website will be live at:

`https://your-username.github.io/repository-name/`

## What to edit

In `index.html`, replace:

`your-email@example.com`

with your real email or course registration link.

You can also replace the brand name, dates, CTA text, and project names directly in `index.html`.
